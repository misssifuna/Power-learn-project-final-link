# Bakulutu Community Hub

### From Empty Tables to The Village Living Room

This project was developed as part of the Power Learn Project AI Safari Capstone.

## Project Overview

Bakulutu Community Hub is a digital transformation proposal for a rural hospitality business. Rather than focusing only on increasing restaurant sales, the project reimagines the business as a community hub that serves different customer groups throughout the day.

The proposed strategy combines hospitality, technology, and community engagement to create a sustainable rural business model.

## Key Features

* Community-focused restaurant concept
* Affordable student lunch programme
* Coffee and bakery experience
* Milk Hub featuring dairy products
* Business Lounge for meetings and workshops
* Technology-enabled operations
* Customer personas and user journey
* Executive summary and strategic recommendations
