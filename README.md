# Bakulutu Community Hub

## From Empty Tables to The Village Living Room

This project was developed as part of the Power Learn Project AI Safari Capstone.

It presents a digital transformation strategy for a rural hospitality business, demonstrating how technology, customer-centered design, and community engagement can transform a traditional restaurant into a sustainable village hub.
